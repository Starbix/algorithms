# Algorithms and Datastructures

## Algorithms

### Sorting

- InsertionSort
  - O(n*log(n)) comparisons
  - O(n²) swaps
- SelectionSort
  - O(n²) comparisons
  - O(n) swaps
- BubbleSort
  - O(n²) comparisons
  - O(n²) swaps
- HeapSort
  - O(n*log(n))
- MergeSort
  - O(n*log(n)) comparisons
  - O(n*log(n)) swaps
- QuickSort
  - O(n*log(n)) average case
  - O(n²) worst-case
