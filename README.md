# Algorithms and Datastructures

## Sorting

- Insertion
  - O(n*log(n)) comparisons
  - O(n²) swaps
- Selection
  - O(n²) comparisons
  - O(n) swaps
- Bubble
  - O(n²) comparisons
  - O(n²) swaps
- Heap
  - O(n*log(n))
- Merge
  - O(n*log(n)) comparisons
  - O(n*log(n)) swaps
- Quick
  - O(n*log(n)) average case
  - O(n²) worst-case
